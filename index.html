Constructions:

  1. Select scenario level and number of characters
  2. Choose a scenario using the button provided or select particular monsters for random scenarios
  3. To create monsters, press on the desired monster type
    • Each monster name has a number - this is the standee that it refers to
    • Tapping on the monster name will toggle between dead/normal/elite and whether or not the monster is a summon (represented by an asterix: *)
    • The individual scenario level for each monster is displayed to the right of the name (you can tap this to change it per monster)
    • Starting health will be the monster's health according to their level
  4. When you're done creating monsters, press the All Active button to show only monsters that are on the board
  5. The blue buttons control the monsters remaining health
  6. When viewing All Active monsters, pressing the monsters name will allow you to toggle the status effects applied to that monster
  7. When a monster is dead, press the Kill Monster button to remove them from the active list
  8. The toggle buttons below the monster healt indicators allow you to apply and remove status effects from monsters in a different way. Toggle on one of the buttons and then press any of the monster buttons to apply the effect.
    • Unselect All will toggle off all of the buttons
    • End Monster Round will remove all but the poison and would status effects from selected monsters (CAUTION: effects applied during a monsters turn (usually strengthen or invisibility) should not be removed this way - see game rules if unsure)
    • Clear All Status Effects will remove all status effects from selected monsters
    • Heal X will apply a healing effect to selected monsters, removing poison and wound and healing damage if applicable (the buttons to the right of this toggle button change the magnitude of the heal)
